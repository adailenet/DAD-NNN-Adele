# DAD-NNN-Adele

Ma pratique s’étend entre plusieurs médiums et plusieurs pensées. J’aime la pratique éditorial, les objets-livres, les livres d’artistes, les livres pauvres. Je peins, je dessine, j’aime confronter mes compétences avec des choses que je ne connais pas. J’ai découvert le code et l’art numérique en deuxième année et j’ai été attiré par leur côté “infini”, où toute idée peut se matérialiser et provoquer des interactions. J’ai souhaité créer une pratique hybride où le matériel et le numérique s'entremêlent, l’un soutient l’autre et inversement. Mes lectures du Manifeste Cyborg de Donna Haraway et de Post-Digital Print d’Alessandro Ludovico ont accompagné cette mutation dans ma pratique “traditionnelle”, glissant progressivement vers des techniques numériques.
